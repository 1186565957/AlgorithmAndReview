# 数据结构及简单算法应用库——不定期更新
## AdvancedStructre
 **用于存储高等数据结构的部分**
 1. 红黑树——BTree（主要依托红黑配色牺牲平衡度，降低部分树枝深度，提高查询速度）
    1. BTree-c为C语言版本实现的红黑树部分。
    2. BTree-c++为C++实现的红黑树部分，内部使用了STL库所以做了区分。
    3. BTreepro为C++版本红黑树的提升版本，因为使用了模板结构，可以储不同的机构。当然如果在C版本中使用void*指针也可以达到相同效果。
 2. 有向图——DirectGraph
